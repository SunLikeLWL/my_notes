# 第一章 Javascrpt简史

javascript是一种脚本语言，通常只能通过Web浏览器去完成一些操作而不能像普通意义上的程序那样独立运行。


DOM是一套对文档的内容进行抽象和概念化的方法。


DHTML（Dynaic HTML，动态HTML）

DHTML不是新技术，而是描述HTML、CSS和Javascript技术组合的术语。

1、利用HTML把网页标记为各种元素；
2、利用CSS设置元素样式和他们的显示位置；
3、利用javascript实时地操控页面和改变样式。
